# gen-ai-sample

python đang chạy: 3.11.13

1. Tạo môi trường ảo:
   python -m venv venv

2. Kích hoạt môi trường ảo:

- Windows:

venv/Scripts/activate

- macOS/Linux:

source venv/bin/activate

3. Cài đặt môi trường:

- pip install -r requirements.txt

4. Chạy app:

- uvicorn main:app --host 0.0.0.0 --port 8000 --reload
